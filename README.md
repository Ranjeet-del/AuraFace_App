# AuraFace

AuraFace is a comprehensive student attendance, monitoring, and placement tracking system built for my 6th-semester project. It is an all-in-one platform designed to sync students, teachers, and administrators by providing seamless management of academic and placement progress. Featuring a native Android mobile application for on-the-go smart attendance (via camera and location) and a robust Python-powered web dashboard for real-time communication and administrative control, AuraFace makes staying connected and tracking academic journeys both easy and efficient.
